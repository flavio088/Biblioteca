# 📚 Biblioteca

Um sistema simples de gerenciamento de livros, construído com Node.js e Express. Permite o cadastro, listagem, atualização e remoção de livros através de uma API RESTful.

## 🚀 Tecnologias Utilizadas

- Node.js
- Express
- PostgreSQL
- Sequelize ORM
- Nodemon
- Dotenv
